# Comfly AI 图像生成器

## 功能

- 支持 `Gemini 3.1` 和 `GPT Image 2` 两个模型页签切换。
- 支持上传参考图，最多可添加 14 张图片。
- 支持点击上传、拖拽上传和 `Ctrl+V` 粘贴图片。
- 支持在提示词中输入 `@` 引用已上传的图片素材。
- `Gemini 3.1` 支持宽高比和分辨率参数设置。
- `GPT Image 2` 支持宽高比参数设置。
- 所有生成任务统一走异步提交与轮询查询。
- 支持任务队列、并发控制和结果预览。
- 支持本地保存、导入、导出 API 配置。
- 支持生成历史记录查看、复用提示词和下载图片。

## 使用方式

1. 直接用浏览器打开 [index.html](/C:/Users/12577/Desktop/comfly-Nano-Banana-App/index.html)。
2. 在“设置与配置”中填写 `API Base URL` 和 `API Key`。
3. 在右侧生成区选择要使用的模型页签。
4. 按需上传参考图，或直接拖拽、粘贴图片到参考图区。
5. 在提示词输入框中编写内容，需要引用素材时输入 `@` 选择对应图片。
6. 根据当前模型设置参数：
   `Gemini 3.1` 可设置宽高比和分辨率。
   `GPT Image 2` 可设置宽高比。
7. 点击“生成图像”，等待异步任务完成。
8. 在结果区查看、放大和下载生成结果。
9. 如需复用历史结果，可在历史记录中再次加载提示词和图片。
